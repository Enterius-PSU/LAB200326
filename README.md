# Задание 1



## Задача 1



### Текст задачи



Решить задачу на построение нового дерева по заданному (map).

Дерево содержит целые числа. К каждому числу приписать заданную цифру в конец.



### Алгоритм решения



1. Запрашиваем количество элементов дерева у пользователя. (Количество должно быть > 0)
2. Создаем бинарное дерево из уникальных чисел с помощью передачи списка использованных чисел.
3. Выводим исходное бинарное дерево.
4. Запрашиваем у пользователя цифру. (Цифра должна быть в диапазоне \[0..9].
5. Создаем новое дерево по заданному условию: каждый элемент нового дерева - это элемент исходного дерева, в конец которого была поставлена заданная цифра. (map)
6. Выводим полученное бинарное дерево.

# Задание 2



## Задача 1



### Текст задачи



Решить задачу на получение заданного результата для указанного дерева (fold).

Сколько элементов дерева не содержит заданную цифру?



### Алгоритм решения



1. Запрашиваем количество элементов дерева у пользователя. (Количество должно быть > 0)
2. Создаем бинарное дерево из уникальных чисел с помощью передачи списка использованных чисел.
3. Выводим исходное бинарное дерево.
4. Запрашиваем цифру для поиска в элементах. (Цифра должна быть в диапазоне \[0..9])
5. Рекурсивно проверяем элементы бинарного дерева на наличие заданной цифры в них. (fold)
6. Выводим полученное количество элементов, не содержащих заданную цифру.
