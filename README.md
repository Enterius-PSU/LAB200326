# Задание 1



## Задача 1



### Текст задачи



Решить задачу на построение нового дерева по заданному (map).

Дерево содержит целые числа. К каждому числу приписать заданную цифру в конец.



### Алгоритм решения



1. Запрашиваем количество элементов дерева у пользователя. (Количество должно быть > 0)
2. Создаем список из случайных чисел с помощью List.map(). (x\*x - 2\*x - 4)
3. По данному списку создаем исходное бинарное дерево.
4. Выводим исходное бинарное дерево.
5. Запрашиваем у пользователя цифру. (Цифра должна быть в диапазоне \[0..9].
6. Создаем новое дерево по заданному условию: каждый элемент нового дерева - это элемент исходного дерева, в конец которого была поставлена заданная цифра.
7. Выводим полученное бинарное дерево.

# Задание 2



## Задача 1



### Текст задачи



Решить задачу на получение заданного результата для указанного дерева (fold).

Сколько элементов дерева не содержит заданную цифру?



### Алгоритм решения



1. Запрашиваем количество элементов дерева у пользователя. (Количество должно быть > 0)
2. Создаем список из случайных чисел с помощью List.fold(). (x\*x - 2\*x - 4)
3. По данному списку создаем исходное бинарное дерево.
4. Выводим исходное бинарное дерево.
5. Запрашиваем цифру для поиска в элементах. (Цифра должна быть в диапазоне \[0..9])
6. Рекурсивно проверяем элементы бинарного дерева на наличие заданной цифры в них.
7. Выводим полученное количество элементов, не содержащих заданную цифру.
